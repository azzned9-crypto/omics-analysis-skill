# 组学富集分析模块

## 前置条件

必须先明确：

1. 特征 ID 类型
2. 输入对象是显著列表还是排序向量
3. 背景集或 universe 规则
4. 富集类型：`ORA` 或 `GSEA`

## 核心规则

1. 未确认 ID 类型前，不得开始富集分析。
2. 必须明确输入是上调、下调、合并列表还是排序向量。
3. 必须记录背景集。
4. 必须记录数据库及其版本信息（若可获得）。
