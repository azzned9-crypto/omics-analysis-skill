# 组学网络分析模块

## 前置条件

必须先明确：

1. 用于网络分析的矩阵
2. trait 表
3. 样本量检查

## 核心规则

1. 未确认样本量是否支持目标解释时，不得直接开展 WGCNA。
2. 必须记录软阈值选择和模块切分参数。
3. 必须同时输出模块归属和模块-性状关联结果。
4. 必须区分探索性模块和已经验证的生物学结论。
